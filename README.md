# ⛩️ Onmyoji Event Bot: Yomihamas Vestiges (Tower Climb) 🎲

Một tool tự động hóa (Bot) bằng Python giúp các đại thần "rảnh tay" cày cuốc sự kiện **Yomihamas Vestiges (Tower Climb) (Onmyoji)**. Bot sử dụng **OpenCV** để nhận diện hình ảnh và **PyAutoGUI** để giả lập thao tác click chuột.

Được tối ưu cho các event dạng **Đổ xúc xắc / Bàn cờ**, giúp tự động vượt ải, đánh boss, nhận quà và xử lý các tình huống bất ngờ trên bàn cờ.

---

## 🌟 Tính năng chính

Tự động nhận diện và xử lý các trạng thái trong sự kiện:
- 🎲 **Tự động đổ xúc xắc** (`btn_dice.png`, `btn_roll.png`).
- ⚔️ **Tự động đánh quái và bỏ qua Boss x3** (`btn_challenge.png`, `btn_challenge_boss.png`).
- 🎁 **Tự động nhận thưởng và đóng hộp quà** (`reward_img.png`, `gift_img.png`).
- ✅ **Nhận diện chiến thắng** để tiếp tục vòng lặp (`victory_img.png`).
- ❓ **Nhận diện Câu hỏi vui** chọn random đáp án (`fun_quiz.png`).
